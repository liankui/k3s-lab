# k3s on Raspberry Pi 5B

## What this guide does

This guide installs a single-node k3s server on the Raspberry Pi 5B at `100.109.40.53`.

## Prerequisites

- Raspberry Pi 5B running Debian 13 (aarch64)
- SSH access as `rpi`
- `sudo` access

## Install

```bash
curl -sfL https://get.k3s.io | sh -
```

## Verify

```bash
sudo k3s kubectl get nodes -o wide
sudo k3s kubectl get pods -A
```

## Uninstall

```bash
sudo /usr/local/bin/k3s-uninstall.sh
```
```

## Troubleshooting
